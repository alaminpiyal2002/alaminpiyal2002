## 🔥 What I've Built

</div>

<br>

### 🏘️ **NearHands — Local Services Marketplace**
> *A full-stack, production-deployed marketplace. Built for real users. Shipped to the cloud.*

A two-sided platform where customers post service requests and local providers respond — covering the entire journey from discovery and negotiation to real-time chat and reputation-building reviews.

- 🔐 **Dual-role JWT auth** — separate dashboards, workflows, and access controls for customers vs. providers
- ⚡ **Real-time layer** via Django Channels + Redis — live chat, typing indicators, read receipts, instant notifications
- 🚢 **Production deployment** across 5 cloud services (Vercel · Render · Neon · Upstash · Brevo SMTP)
- 🔁 **CI/CD pipeline** with GitHub Actions running automated checks on every push
- 📖 **Full API docs** via Swagger/OpenAPI + Postman collection across 3 environments (local · Docker · production)

`Django REST Framework` `React` `Tailwind CSS` `PostgreSQL` `Redis` `WebSockets` `Docker` `JWT` `GitHub Actions`

<br>

### 💰 **EXPORA — Expense Tracking Web App**
> *Personal finance, done right — isolated, insightful, and exportable.*

A full-featured finance tracker built with per-user data isolation, a live analytics dashboard, and CSV export — deployed on Render with a fully responsive UI.

- 🔒 **Complete data isolation** — every user sees only their own income, expenses, and categories
- 📊 **Auto-calculated analytics** using Django ORM aggregations — monthly breakdowns without any manual math
- 📤 **CSV export** so users can take their data anywhere

`Django` `Bootstrap 5` `SQLite` `Django ORM` `Token Authentication`

<br>

### 👁️ **Eye Disease Detection — Web-Based Deep Learning App**
> *AI-powered medical imaging accessible from any browser.*

A web app that classifies 6 eye diseases from retinal photos — no specialist hardware, no heavy infrastructure. Upload an image, get a diagnosis-ready prediction in real time.

- 🧠 **Knowledge distillation** — lightweight StudentCNN, fast inference on 224×224 RGB images without a GPU
- 🔗 **End-to-end Django pipeline** — image upload → PyTorch inference → live result rendering in one app

`Django` `PyTorch` `CNN` `Knowledge Distillation` `Computer Vision` `NumPy`
